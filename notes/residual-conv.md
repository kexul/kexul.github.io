# 残差卷积

Residual Connection（残差连接）是深度学习中最常用的技巧之一。

## 核心思想

不是让网络直接学习映射 `x → y`，而是学习残差 `x → y - x`，即：

```
output = F(x) + x
```

其中 `F(x)` 是卷积层学习的内容，`x` 是跳连过来的原始输入。

## 为什么有效

- 梯度可以直接通过跳连反向传播，缓解梯度消失
- 网络深度可以大幅增加（ResNet 能做到 100+ 层）
- 浅层特征不会丢失，深层网络也能保留原始信息

## 在 DDPM/Stable Diffusion 中的应用

DDPM 的 UNet 把普通卷积替换成了**带残差的卷积块**（ResBlock），使得：
- 去噪网络可以训练得更深更稳定
- 时间步嵌入（timestep embedding）也通过这个结构注入
